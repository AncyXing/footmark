    辣椒炒肉

    vuePress-theme-reco    2023

    辣椒炒肉

    # 厨具

    1. 电磁炉
    2. 平底锅

    # 食材

    1. 青椒
    2. 大蒜
    3. 姜
    4. 豆豉
    5. 猪肉

    # 调料

    1. 盐
    2. 生抽
    3. 料酒
    4. 耗油
    5. 油
    6. 淀粉

    # 其他工具

    勺子：用来衡量调料用量

    # 关键点

    # 1. 猪肉要进行腌制

    腌制后的口感会更好

    # 腌制用量

    • 1小勺盐
    • 2勺酱油
    • 1勺耗油
    • 2勺料酒
    • 2勺淀粉

    猪肉逆纹理切好放入大碗中，加入上面的调料后抓匀，抓匀后再加上2勺油，再次抓匀放置10分钟左右，这段时间可以准备辅料。

    # 2. 辣椒干煸

    热锅不加油，放入斜刀切好的辣椒，大火干煸至虎皮状后，放入半勺盐，炒匀后盛出备用

    # 操作步骤

    1. 洗菜：将青椒、姜、剥好的大蒜、猪肉用清水清洗干净
    2. 切菜：大蒜切成蒜末、姜切成姜末、青椒斜刀切成马蹄状、猪肉逆纹理切成肉片
    3. 洗锅并将锅擦干
    4. 辣椒干煸后盛出(1分钟)
    5. 不用洗锅，开中火，往锅中加入2勺油，热后倒入腌制好的猪肉
    6. 猪肉炒至变色后盛出(2分钟)
    7. 不用洗锅，开中火，往锅中加入1勺油，热后倒入蒜末、姜末炒香(15秒)
    8. 加入猪肉翻炒均匀(1分半钟)
    9. 加入辣椒翻炒均匀(1分钟)
    10. 加入1勺豆豉 、1勺生抽翻炒均匀(30秒)
    11. 出锅装盘
